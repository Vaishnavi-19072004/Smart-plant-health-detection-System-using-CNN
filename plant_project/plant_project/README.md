# Smart Plant Health Detection System using CNN

## 🌿 Project Overview
This project uses a Convolutional Neural Network (CNN) to detect plant leaf diseases from images. It classifies leaves as **healthy or unhealthy**.

---

## 🚀 Features
- Plant disease detection using AI
- Image-based classification
- CNN deep learning model
- Simple prediction UI

---

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- CNN (Convolutional Neural Network)
- OpenCV
- NumPy
- Tkinter
